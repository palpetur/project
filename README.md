# project
About the project
Our project is website, where you can download a game. This game is a 2D platformer/puzzle game.
The website has a login feature, so the player can play with their same level on other devices. But you can play without login it. For creating the website, we will use Visual Studio Code.
The website's most used colors will be blue yellow, dark grey. When you go to main page, you will see a login and a download feature. For the login you create a nickname and a password. In the database the nickname will be unique, so you can't use a name, which someone already has. After the login, the website will show your stats, levels you finished, skins you have and some other things. It will show a toplist too where can search anyone and see their scores. The toplist automaticaly shows your place.
The game will be created in unity or netbeans. And Github both for website and game.
The gameplay is you controll a fox, must avoid hunters and bears, find a house and steal chickens. In this game you can unlock new levels and skins like arctic fox, desert fox or kitsune.
In the harder levels you also have to protect kid foxes, who came with you.
